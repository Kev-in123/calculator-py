# Calculator.py

This package is a simple calculator.

## Installation

Stable version:

```
pip install caculatorpy
```

Working version:

```
pip install git+https://github.com/Kev-in123/caculator-py
```

## Example Usage

```python
from caculatorpy import Calc

c = Calc(5, 6)
print(c.add())
```
